# Student Grade Analysis Project

## Project Overview

This project analyzes student academic performance using a student grade dataset.  
The goal is to explore how different demographic, family, school, and study-related factors may be associated with students' final grades.

The analysis includes data cleaning, exploratory data analysis, correlation analysis, hypothesis testing, effect size analysis, and confidence interval estimation.
